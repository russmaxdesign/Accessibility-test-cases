Select menus and keyboard use

Operation

Windows

  • TAB to focus on select menu
  • Focus appears on select menu
  • DOWN ARROW move to next option, UP ARROW to move to previous option
  • SPACEBAR to expand (make all optons visible)
  • ESC to expand options (make all optons visible) - Windows only
  • option chosen using ARROW only

OSX

  • TAB to focus on select menu
  • Focus appears on select menu
  • DOWN ARROW move to next option, UP ARROW to move to previous option
  • DOWN ARROW and UP ARROW automatically expand options
  • Option chosen using ENTER or SPACE BAR.
